# Huawei Zheng

Personal academic website of Huawei Zheng, a master student in Software Engineering at Zhejiang University.

Website: <https://huawzheng.github.io>

Built with [Jekyll](https://jekyllrb.com/) and the [al-folio](https://github.com/alshedivat/al-folio) theme.

## Local Development

```bash
bundle exec jekyll build --baseurl ""
bundle exec jekyll serve --host 127.0.0.1 --port 4000 --baseurl "" --no-watch
```
