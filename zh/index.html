我是郑华伟，目前为浙江大学软件工程硕士研究生，导师为邓达臻老师，预计 2027 年毕业。

我的研究方向围绕数据智能体与可信大语言模型，重点关注 LLM 安全、检索推理、智能问答系统、Text-to-SQL 与 Schema Linking 等方向中的可靠系统构建。

研究之外，我喜欢电影、剧集、阅读、音乐、历史、科技和游戏，尤其是能带来启发或沉浸体验的作品。成功只有一个——按照自己的方式，去度过人生。——《明朝那些事儿》

你可以通过 zhenghuawei [at] zju.edu.cn 联系我，也可以在 GitHub 和 Google Scholar 查看我的更多工作。
